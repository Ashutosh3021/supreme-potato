# Portfolio Website

This is a cinematic portfolio website for Ashutosh Patra built with Next.js, featuring dramatic visual styling inspired by Batman/God of War/Game of Thrones with bold typography and strong visual hierarchy.

## Technology Stack

- Next.js 16.1.1
- React 19.2.3
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- Custom Interactive Cursor Effects
- Cinzel Decorative Font

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

### GitHub Pages

To deploy this portfolio on GitHub Pages:

1. Create a GitHub repository at https://github.com/Ashutosh3021/PORTFOLIO--WEBSITE
2. Push your code to the repository
3. In your repository settings, go to Pages section
4. Select source as GitHub Actions
5. Create a `.github/workflows/deploy.yml` file with the proper configuration
